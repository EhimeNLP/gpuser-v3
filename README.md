## Usage

### 環境変数の設定

```bash
cp .env.example .env
cp backend/.env.example backend/.env
```

### 起動

.envファイルで設定したポート番号でアクセスできます（デフォルト：<http://localhost:8000>）

```bash
make deploy
```
